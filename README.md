# LLM Hardware Matcher

Спокойный статический помощник, который отвечает на один вопрос: **какие локальные LLM запустятся на моем железе и как их открыть через Ollama или LM Studio?**

## Что уже есть

- выбор типа устройства: Windows/Linux PC, Mac, handheld, iPhone/iPad, Android и CPU-only;
- короткий выбор железа без OEM-названий: например, `RTX 3060` → `12 GB`;
- режимы подбора: быстро, баланс, максимум качества, «лишь бы запустить»;
- расчет примерной памяти модели с учетом кванта, контекста и GPU + RAM offload;
- карточки рекомендаций с понятным статусом, квантованием, контекстом и командой `ollama run`;
- осторожные подсказки для AMD, Intel, handheld и мобильных устройств.

## Запуск

```bash
npm install
npm run dev
```

Для production-проверки:

```bash
npm run build
npm run preview
```

## Структура

```text
src/
  App.tsx                  # основной экран выбора и результатов
  data/
    hardware.ts            # seed-база железа
    models.ts              # seed-база моделей
    quantization.ts        # кванты GGUF/Ollama/LM Studio
  lib/
    calculator.ts          # упрощенный расчет совместимости
    format.ts              # форматирование статусов и памяти
  styles/globals.css       # спокойный UI в духе концепции
```

## Ограничения MVP

Расчет намеренно приблизительный: он помогает выбрать практичный вариант, но не обещает точность до байта или стабильные tok/s. Спеки новых GPU и телефонов нужно сверять перед расширением базы.
